- §22.3 Catalog/Import: `SKU_DUPLICATE`, `BARCODE_DUPLICATE`, `CATEGORY_CYCLE`, `PRODUCT_ARCHIVED`, `COST_PERMISSION_REQUIRED`, `IMPORT_FILE_INVALID`, `IMPORT_MAPPING_INVALID`, `IMPORT_PREVIEW_STALE`, `IMPORT_JOB_STATE_INVALID`, `IMPORT_APPLY_FAILED`.
