# @ai-sales/i18n

Locale, ICU message catalog, and date/number/currency formatting (spec 5.1, 7.6), built on
`react-intl`.

- `vi-VN` is the baseline locale (`DEFAULT_LOCALE`); `SUPPORTED_LOCALES` is currently
  `["vi-VN", "en-US"]`.
- No README; the baseline-locale note is an inline comment in `AppIntlProvider.tsx`.
